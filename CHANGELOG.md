# Changelog

Tabloom 的重要变化记录在此文件中。版本号遵循语义化版本规则。

## [2.1.2] - 2026-07-14

### Changed

- 提高 Subtle 和 Playful 两档声音的整体增益。
- 进一步加强单项关闭和批量关闭的瞬态、低频和收尾声音。

### Fixed

- 修复网站卡片最右侧操作按钮的 tooltip 被卡片边界裁切的问题。
- 行内 tooltip 改为向上弹出，最右侧提示向左对齐，并在 hover/focus 时提升卡片层级。

## [2.1.1] - 2026-07-14

### Added

- 完成、收藏、关闭、撤销和折叠等关键操作的点击位置光圈与粒子反馈。
- 为键盘触发操作提供以控件中心为起点的光标效果。

### Fixed

- 提前在用户按下指针时初始化 Web Audio，修复第一次关闭网页或确认批量关闭后没有声音的问题。
- 加强关闭和批量关闭音效的瞬态与音量，使 Subtle 模式下仍能清晰感知。

## [2.1.0] - 2026-07-14

### Added

- `Off`、`Subtle`、`Playful` 三档声音设置，默认使用 Subtle。
- 完成待处理、加入收藏、加入稍后处理、关闭标签页、网站级处理、批量清理、撤销和折叠的触感动效。
- 基于 Web Audio API 的本地声音反馈，不增加网络请求或音频素材依赖。
- 批量操作的错峰卡片退出效果和单次组合声音。
- 对系统“减少动态效果”设置的支持。

### Changed

- 声音偏好保存到本地，并兼容 2.0.0 版本已有数据。

## [2.0.0] - 2026-07-14

### Added

- 全新的 Tabloom 桌面端管理页面。
- For Later、Open Tabs、Saved 三个纵向业务区块。
- 按网站聚合标签页和网页 favicon/logo。
- 读取网页自身的标题、description 摘要与站点名称。
- 标签页切换、单项关闭、网站级关闭和 Close all。
- 可在标签页关闭后继续保留的 For Later。
- 本地持久化 Saved 收藏。
- 搜索、区块折叠、批量确认、单步撤销和操作反馈。
- System、Light、Dark 三种主题模式。

### Changed

- 产品从旧版 `Tab Atlas / 标签星图` 更名为 `Tabloom`。
- 从 420px 工具栏弹窗改为完整的浏览器桌面页面。
- 产品 UI 统一使用英文，网页内容保持原始语言。

### Known limitations

- 受保护的浏览器内部页面无法读取 description。
- 关闭标签页后的撤销不能恢复原浏览历史。
- 当前数据只保存在本地，不支持跨设备同步。
